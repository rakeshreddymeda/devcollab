# DevCollab

Real-time developer collaboration platform built using the MERN stack.

## Features

- User authentication (JWT)
- Project management
- Kanban task board
- Real-time updates with Socket.io

## Tech Stack

Frontend
- React
- Axios
- Socket.io Client

Backend
- Node.js
- Express
- MongoDB
- JWT

## Deployment

Frontend: Vercel  
Backend: Render  
Database: MongoDB Atlas
