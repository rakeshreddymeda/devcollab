# DevCollab

Real-time developer collaboration platform built using the MERN stack.

## Features

- User authentication (JWT)
- Protected routes
- React Context auth
- Project collaboration
- Kanban task board (drag & drop)
- Real-time task updates with Socket.io
- Task comments
- Environment config
- Error handling
- Production deployment

## Tech Stack

Frontend
- React
- Axios
- Socket.io Client

Backend
- Node.js
- Express
- MongoDB
- JWT

## Deployment

Frontend: Vercel  
Backend: Render  
Database: MongoDB Atlas
